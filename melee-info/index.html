Описание

На этой странице собрано холодное оружие YourWorld по категориям получения и подкатегориям. Для каждой модели можно посмотреть скорость разделки, обычный и сильный удар, а также разделку по радиусу, если этот параметр есть у оружия.

Если у модели есть несколько вариантов внешнего вида, они переключаются стрелками прямо в карточке. Изображение оружия можно открыть в увеличении.

Ачивки

Названия достижений и что нужно сделать для их выполнения

Описание

На сервере есть система достижений, которые выполняются за разные активности: фермерство, путешествия, транспорт, медицина, торговля, PvE, поиск тайников и другие действия.

Ниже собраны названия ачивок, их иконки из мода и прогрессия по этапам, если у достижения есть несколько уровней.

Телепорты

Перемещение между трейд-зонами через терминалы и специальные карточки

Как это работает

  • На каждой трейд-зоне установлен терминал для перемещения между трейд-зонами.
  • С помощью терминала можно быстро добраться до другого трейда, если у вас есть нужная карточка для телепорта.
  • Карточки для телепорта можно приобрести на любом трейде.
  • Нужную точку перемещения удобно подготовить заранее, если вы часто перемещаетесь между трейдами.

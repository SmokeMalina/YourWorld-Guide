Стол для покраски

Перекраска предметов и долларовых сетов

Описание

Стол для покраски используется для перекраски различных предметов. Сейчас через него перекрашиваются только долларовые сеты, но в будущем стол будет пополняться новыми покрасками и для других предметов.

Сам стол можно скрафтить или использовать уже установленный стол на трейд-зоне.

Для покраски понадобится предмет, который вы хотите перекрасить, и баллончик для применения выбранного варианта.

Как использовать

Подготовка

  • Найдите или скрафтите стол для покраски.
  • Возьмите нужный предмет для перекраски.
  • Подготовьте баллончик.

Процесс

  • Положите внутрь стола предмет, который хотите покрасить.
  • Откройте меню стола.
  • Выберите нужную покраску из доступных вариантов.
  • Примените покраску с помощью баллончика.

Важно

  • Сейчас перекрашиваются только долларовые сеты.
  • Список доступных покрасок будет расширяться.
  • Если для предмета нет варианта покраски, он не появится в меню.

Как зайти на сервер

Подключение к серверу через пресет DayZ Launcher

На этой странице собрана пошаговая инструкция, готовый файл пресета и видео, по которому можно быстро повторить все действия.

Инструкция

1. Скачайте пресет

  • Для входа на сервер скачайте файл DayZ Mod Preset YourWorld.html.
  • Не переименовывайте файл: расширение должно остаться .html.

2. Импортируйте пресет в DayZ Launcher

  • Откройте обычный DayZ Launcher из Steam.
  • Перейдите во вкладку MODS.
  • Импортируйте пресет любым удобным способом.
  • Можно выбрать Ещё → Импортировать список модов из файла и указать скачанный пресет.
  • Либо просто перетащите файл мышкой в окно лаунчера.

3. Установите моды

  • Лаунчер предложит установить моды из пресета.
  • Нажмите Подписаться / Установить на всё.
  • Дождитесь, пока Steam докачает все нужные моды.

4. Включите пресет

  • Во вкладке Моды → Настройка выберите пресет YourWorld.

5. Зайдите на сервер

  • Можно зайти через список серверов.
  • Если пресет уже загружен, нажмите Играть: сначала откроется меню, а затем вы сможете зайти на нужный сервер.

Если не пускает

Лаунчер просит моды

  • Скорее всего, моды ещё не докачались.
  • Вернитесь во вкладку Моды и дождитесь, пока Steam завершит загрузку.

Если что-то зависло

  • Перезапустите Steam и DayZ Launcher.
  • После перезапуска снова выберите пресет YourWorld.
